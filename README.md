# Fact API Service

This project provides a simple RESTful API for fetching interesting facts. It's built to offer quick access to a collection of facts through well-defined endpoints.

### Features
*   **RESTful API:** Exposes endpoints to retrieve factual information.
*   **Random Fact Retrieval:** Easily fetch a single random fact.
*   **OpenAPI Specification:** Comprehensive API documentation available via `openapi.yaml`, detailing all endpoints, request/response structures, and authentication (if applicable).
